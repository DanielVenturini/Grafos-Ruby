# Grafos-Ruby

Como estou começando a aprender a linguagem Ruby, estou criando um grafos para ir praticando.
O grafo já tem as funções basicas de inserir, printar e duas de busca: largura; e profundidade. Tambem tem a função de ordenação topológica, porem a mesma depende da ordem em que os vertices e arestas são inseridos. Em qualquer ordem ela funciona, porém o resultado varia. Agora está adicionado a classe Aresta, assim sendo possivel implementar alguns algoritmos para gerar a arvore minima.
